# Sentiment Analysis

We are using LSTM model to train sentiment analyser over twitter data. 
The idea is to extract the tweets first 
 
### Execution Flow
1. Extract data using from NodeJs App (Data Extraction using Twitter API)
1. Place your data in data folder
2. Pre-process data using sentiment-analysis/data-pre-processing.py
3. Train model using sentiment-analysis/sentiment-analysis-LSTM.py

#### Dependencies
1. TensorFlow >= 2.0
